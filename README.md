# JLPKangaroo_OW_OT
A modified version of JLP Kangaroo that allows you to generate only wild or only tame kangaroos.  
Make a replacement in Constants.h file to produce only tame kangaroos:   
//#define ONLY_WILD  
#define ONLY_TAME  
or to produce only wild kangaroos:   
#define ONLY_WILD  
//#define ONLY_TAME
